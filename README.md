# Watchtower — Official Website

**Watch. Read. Everything.**

This is the source code for the official [Watchtower](https://github.com/ferelking242/watchtower) website, built with [VitePress](https://vitepress.dev/).

Watchtower is a full-featured anime and manga reader/player for Android, giving you access to hundreds of sources through its powerful extension system.

## Stack

- [VitePress](https://vitepress.dev/) — Static site generator
- [Vue 3](https://vuejs.org/) — UI components
- [Element Plus](https://element-plus.org/) — UI library

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 20
- [pnpm](https://pnpm.io/) >= 8

### Installation

```bash
pnpm install
pnpm dev
```

The site will be available at `http://localhost:5173`.

### Build

```bash
pnpm build
```
